### ✦ About Me

Technical Program Manager at **Studio Node** and Software Engineering senior at **Utah Tech University**, graduating May 2026.

I started my degree thinking I'd become a developer. What I found was that I'd rather coordinate the people building software than write the code myself — bridging technical teams and clients, gathering requirements, and keeping cross-functional work on track.

Brazilian, bilingual in English and Portuguese, and always up for conversations about UX, program management, and building things people actually want to use.
